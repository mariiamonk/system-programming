# Системное программирование
### О проектах
В рамках курса по системному программированию были написанны 4 лабораторные работы.
| Номер                  | Название   | Описание       |
|---------------------------|--------------------|--------------|
| 1 | Утилита sort для Linux| Реализация аналога стандартной утилиты sort на языках C и Assembly. Проект демонстрирует навыки работы с системными вызовами Linux напрямую, без использования стандартной библиотеки ввода-вывода (stdio.h). |
| 2 | Драйвер PCIe-устройства для ОС Linux  | Данный проект представляет собой реализацию символьного драйвера устройства для ОС Linux, взаимодействующего с виртуальным PCIe-устройством в среде QEMU. Устройство эмулирует работу буфера ввода данных с хостовой машины. |
| 3 | Ядро | Лабораторная работа по разработке минимального ядра операционной системы, совместимого со спецификацией Multiboot2. Программа загружается через GRUB2 в среде UEFI (OVMF) и извлекает системную информацию (ACPI), переданную загрузчиком. |
| 4 | Загрузчик ядра | Разработка собственного загрузчика операционных систем на языке C с использованием фреймворка EDK2. Приложение работает в среде UEFI (OVMF) и обеспечивает загрузку ядер, соответствующих спецификации Multiboot2. |
